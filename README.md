# 熟悉几种分治算法的设计与实现

### [上机目的]
#### 1、掌握分治的基本思想。
#### 2、熟练掌握一些基本结构的遍历算法。
### [上机要求]
按以下上机内容完成各题目，并注意观察编译、运行过程中出现的问题以及记录解决方法。
### [上机内容]
采用分治法来设计棋盘覆盖、特定二分搜索、自然合并排序或随机快速排序（两个排序算法二选一）：
（1）棋盘覆盖：在一个2k*2k个方格组成的棋盘中,恰有一方格残缺, 残缺方格的位置有22k种。故对任何k≥0,残缺棋盘有22k种. 要求用4种L型骨牌覆盖残缺棋盘上的所有方格，且任何2个L型骨牌不得重叠覆盖。求L型骨牌的覆盖方案。\
实现要求：可以设定棋盘的方格行数和残缺方格的位置（棋盘行号和列号）；要求输出棋盘覆盖结果参见课件PPT。（棋盘以二维数组存储，每个方格对应一个数组元素通过赋相应数值表示覆盖结果，特殊方格对应的数组元素以数值1表示，每次用L型骨牌覆盖方格通过对应数组元素赋以相同数值表示，按照分治法的L型骨牌覆盖次序，对应数组元素的赋值从2开始递增）\
（2）特定二分搜索：给定一个有序正整数数组和一个要查找的目标数。要求给出要查找目标数在给定数组中第一次(或最后一次)出现的索引号（数组下标），没有查找到时输出-1。\
实现要求：要求能够执行中人工输入数组和目标数实例，之后输出目标数第一次(或最后一次)出现的索引号。\
（3）自然合并排序或随机快速程序（二选一）：排序问题设为对一组数值从小到大排序，要求采用自然合并排序或随机化快速排序两种分治思路实现。\
实现要求：可以任意给定一组不同个数、不同数值的数据；输出排序后结果和排序时间。
### [检查标准]
（1）算法实现可正确在机器上执行并按要求输出正确结果；\
（2）能够正确指出并讲解核心部分代码。
